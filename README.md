# Online Food Delivery Management System

## Overview
This project is an **Online Food Delivery Management System** developed as part of an **Information Systems module**. The system simulates a real-world food ordering and delivery platform, enabling users to browse menu items, place orders, and manage food selections while supporting efficient food delivery operations.

## Objective
The objective of this project is to understand how information systems support real-world operations by designing and implementing a functional food ordering and delivery system.

## Key Features
- Browse available food items  
- Order Regular and Combo meals  
- Dynamic menu management  
- Organized food categories  
- Price tracking with history (stores previous and updated prices with timestamps)  
- Efficient order handling  
- Basic support for food delivery workflow  

## Technologies Used
- **Java** (Backend development)  
- **HTML** (Structure)  
- **CSS** (Styling)  
- **JavaScript (basic)** (Client-side interaction)

## System Workflow
1. Users browse available menu items  
2. Select meals (Regular or Combo)  
3. System processes the order  
4. Menu items and pricing are dynamically managed  
5. Order flow supports food ordering while integrating with menu and pricing components
